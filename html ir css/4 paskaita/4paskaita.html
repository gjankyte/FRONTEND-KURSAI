terminalo komandos

  • cd - change directory - pakeisti direktorija (vaiksciojimas tarp aplankalu)
  • ls - list files and directories - isvardinti failus ir direktorijas.
  • CTRL C - atsaukia vykdoma veiksma.
  • TAB - uzbaigia fraze arba pateikia pasiulymus.
  • clear - nukelia/paslepia terminale rodoma info i virsu.
