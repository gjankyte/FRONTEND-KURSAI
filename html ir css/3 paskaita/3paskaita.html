Puslapio struktura

  • header - puslapio virsus kuriama nurodoma informacija apie puslapi / meniu
  • nav - skirtas aprasyti/kurti meniu
  • section - didzioji puslapio dalis
  • div - mazoji puslapio dalis
  • main - pagrindine puslapio dalis
  • article - main viduje esanti sekcija
  • aside - sonine puslapio sekcija
  • footer - puslapio apacia, kurioje yra ivairi informacija kaip: terms and uses; privacy policy; copyrights; links; partners...

Menu Info

Kuriant menu naudojamas 'nav' tagas, kurioje viduje yra rikiuotas/nerikiuotas sarasas. To saraso elementu viduje yra nuorodos tagas (a). Kuris nurodo i kazkuria puslapio dali arba kita puslapi.

Nuorodos tago (a) atidaromojoje dalyje esantis atributas href nurodo i kur vartotojas turi buti nukeltas paspaudus ant nuorodos.

Jeigu atributas nurodo i kazkoki ID (href="#ID_pavadinimas"), tuomet vartotojas yra nukeliamas i puslapio dali kuriai yra suteiktas nurodytas ID (id="ID_pavadinimas").
Pvz.:puslapio virsus

Jeigu href atributas nurodo i kita html faila (href="./aplankas/failas.html"), tuomet vartotojas yra nukeliamas i nurodyta html faila.
Pvz.: Kitas puslapis

Inline Block ir InLine-Block

Inline

InLine elementai yra tokie elementai kurie uzima tik tiek vietos ekrane kiek jiems yra reikalinga. Minusas tas kad negalima keisti siu elementu dydziu. Pliusas tas kad jie labai lengvai italpinami tarp kitu informacijos vienetu.
Tagu pavydziai: a, span, img, input, button, label, select, textarea, cite..

Block

Block elementai yra tokie elementai kurie uzima visa leistina ploti ir neisileidzia kitu elementu i ta pacia eilute. Minusas tas kad uzima visa leistina ploti. Pliusas tas kad galima keisti ju dydi.
Tagu pavydziai: h1-h6, p, div, section, main, article, nav, ol, ul, li, blockquote, aside, canvas, table..

Inline-Block

InlineBlock elementai yra tokie elementai kurie pasisavino geriausias Blockiniu ir Linijiniu elementu savybes. Jie uzsima tik kiek vietos jiems reikia, bet ju dydi vis viena galima keisti.

Model box

Margin - atstumas nuo elemento remeliu iki aplinkiniu elementu remeliu( skirtingu elementu marginai persidengia ties virsum/apacia, ne ties sonais).

Border - Remelis kuris yra tarp Margin ir Padding.

Padding - atstumas nuo elemento content iki elemento remelio.

Content - elemento turinys. (blockinio elemento turinys, uzims visa leidziama eilute arba nurodyta dydi)
