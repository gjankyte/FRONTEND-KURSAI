parent / child elements

elementas kuris savo viduje turi kitus elementus - tevinis (parent) elementas.
Elementas kuris yra kazkokio elemento viduje - vaikinis (child) elementas.

Elementas gali tureti viena vieninteli tevini elementa.
Elementas gali tureti daugeli vaikiniu elementu.

CSS selectors

Visus CSS selectorius galima rasti Cia.

  • Element - tago vardas
  • .class - tagui suteikta klase
  • #id - tagui suteiktas id
  • .class1.class2.class3 - tagui suteiktos abi(arba daugiau) klases
  • .class3 .class4 - tagui suteikia stilius, jeigu jis turi class4 klase ir yra class3 palikuonis.
  • * - parenka viska
  • element1, element2, - isvardintus ir kableliu atskirtus elementus
  • elements1 > element2 : element2, kurio tevinis elementas yra elemen1
  • a:link - nelankytos nuorodos
  • a:visited - lankytos nuorodos
  • a:active - aktyvi nuoroda (paspaudimo momentas)
  • element:hover - elementas ant kurio yra uzvesta pele
