CSS rasymo budai

  • Inline - linijinis rasomas tago viduje naudojant style atributa
  • Internal - Vidinis rasomas style tago viduje
  • External - Isorinis rasomas atskirame CSS faile, kuri reikia prijungti prie HTML naudojant link taga

CSS selectoriai

  • element - parenkamas pagal jo pavadinima (p{})
  • id- sukuriamas susigalvotas id pavadinimas ir priskiriamas vienam elementui (#idPavadinimas{})
  • class- sukuriamas susigalvotas klases pavadinimas ir priskiriamas bet kokiam kiekiui elementu (.klasesVardas{})

Stiliai

  • color - skirtas pakeisti teksto spalva (color: red)
  • font-size - skirtas pakeisti teksto dydi (font-size: 20px)
  • background-color - skirtas pakeisti fono spalva (background-color: aqua)
  • font-family - skirtas pakeisti teksto srifta (font-family: monospace)
  • Font-weight - skirta pakeisti teksto storuma(font-weight: bold)
  • line-height - skirtas pakeisti teksto eilutes auksti (line-height: 20px)
  • border - skirtas nurodyti remelio stori+tipa+stori (border: 5px dotted/dashed/inline/outline.. solid black)
  • border-radius - skirtas apvalinti remelio kampus (bordder-radius: px,em,%)
  • text-align - skirtas lygiuoti teksta (text-align: center)
  • font-style - skirtas suteikti stiliu tekstui (font-style: italic)
